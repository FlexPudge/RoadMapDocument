Если ссылка - это псевдоним объекта, то указатель - это переменная, которая содержит адрес объект. Указатель - это буквально указка: он не владеет значением сам по себе, он лишь показывает, где это значение лежит в памяти. В отличии от ссылок, указатель можно не только переназначить, но и сделать пустым - т.е. нулевым nulltrp.
Через `*p` можно разыменовать указатель и прочитать значение, которое лежит по этому адресу.  

```
int a =5;
int* p = &a; // A. захват адреса
*p = 7; //B. разыменование 
p = nullptr; //C. переназначение
```

Что лежит в `p`? - Машинный адрес + provenance-тег
Можно ли писать через `const int*`? - Нельзя, тк `const` защищает _pointed-to_ объект
