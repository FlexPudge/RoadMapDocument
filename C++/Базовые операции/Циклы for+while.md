Циклы позволяют выполняет некоторый набор инструкций множество раз, пока соблюдается определенное условие. В языке C++ имеются следующие виды циклов:
- for
- while
- do while
**For**
Цикл while выполняет некоторой код, пока его условие истинно, то есть возвращает true. Он имеет следующее формальное определение:
```
while(условие)
{
	// выполняемые действия
}
```

После ключевого слова while в скобках идет условное выражение, которое возвращает true или false. Затем в фигурных скобках идет набор инструкций, которые составляют тело цикла.
И пока условие возвращает true, будут выполняться инструкции в теле цикла.
