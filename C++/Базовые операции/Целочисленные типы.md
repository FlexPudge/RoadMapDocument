- Целочисленные типы записываются несколькими ключевыми словами
	- Signed/unsigned  - обозначают знаковость
	- short/long  - обозначают размер
	- int - тип

**Signed** - указывает, что переменная может принимать как положительные, так и отрицательные значения. При этом левый бит области памяти, выделяемый для хранения значения, используется для предоставления знака:
- Если этот бит установлен в 0, то значение переменной считается положительным.
- Если бит установлен в 1, то значение переменной считается отрицательным.
Применяется к типам char, short, int, long, long long.
По умолчанию тип int - signed.
**Unsigned** - указывает, что переменная принимает неотрицательные значения. При этом самый левый бит области памяти, выделяемый для хранения значения, используется так же, как и все остальные биты области памяти - для предоставления значения.

**Важно**: при назначении отрицательного значения переменной типа unsigned значение преобразуется в дополнение по модулю 2, потому что unsigned-типы не могут хранить отрицательные значения. 

**Важно**: компиляторы предупреждают о сравнениях между signed и unsigned типами, потому что диапазоны значений этих типов варьируются. Это может происходить, например, когда отрицательное число преобразуется в unsigned-тип или положительное число — в signed.

Для индексации в массиве нужно использовать тип size_t
